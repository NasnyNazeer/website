Intro to HTML

  • Complete the Learning Lab Course - Intro to HTML
  • Add some information about yourself on your website using HTML
  • Post a message on ATFG Discord #students channel with the hashtag #intro-to-html (see example message below)
  • fun-fact: chit chat in the #students channel can earn you karma points.

Example Message

Hi! I've completed the Intro to HTML course.
My repository URL is https://github.com/rajeev/intro-html and my website URL is rajeev.github.io/intro-html
#intro-to-html

(You can also post your own custom message with your repository URL and the hashtag #intro-to-html)
